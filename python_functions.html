01 — Основы

Что такое функция?

Функция — это именованный блок кода, который можно вызывать многократно. Она принимает входные данные (аргументы), выполняет действия и возвращает результат.

Ключевое слово def объявляет функцию. Блок кода внутри выполняется только при вызове.

hello.py 
# Объявление функции
def greet(name):
    print(f"Привет, {name}!")

# Вызов функции
greet("Сергей")   # → Привет, Сергей!
greet("Мир")     # → Привет, Мир!

02 — Аргументы

Виды аргументов

Позиционные и именованные

Позиционные передаются по порядку, именованные — по ключу. Можно комбинировать.

args.py 
def power(base, exp=2):          # exp — со значением по умолчанию
    return base ** exp

power(3)          # → 9   (exp=2 по умолчанию)
power(3, 3)       # → 27  (позиционно)
power(exp=4, base=2)  # → 16  (именованные)

*args и **kwargs

*args собирает произвольное число позиционных аргументов в кортеж. **kwargs — именованные в словарь.

varargs.py 
def total(*nums):
    return sum(nums)

total(1, 2, 3, 4)   # → 10

def info(**data):
    for k, v in data.items():
        print(f"{k}: {v}")

info(name="Сергей", city="Минск")
# name: Сергей
# city: Минск
03 — Возврат значений

return и None

Функция без return возвращает None. Можно вернуть несколько значений сразу — Python упакует их в кортеж.

return.py 
def divide(a, b):
    if b == 0:
        return None, "Деление на ноль!"
    return a / b, None

result, error = divide(10, 2)
print(result)   # → 5.0

result, error = divide(10, 0)
print(error)    # → Деление на ноль!
04 — Лямбда

Лямбда-функции

Однострочные анонимные функции. Удобны для сортировки, фильтрации, map/filter.

lambda.py 
# Обычная vs лямбда
def square(x): return x**2
sq = lambda x: x**2

# Сортировка по ключу
students = [("Аня", 19), ("Боря", 22), ("Вася", 17)]
students.sort(key=lambda s: s[1])
# [("Вася", 17), ("Аня", 19), ("Боря", 22)]

# map и filter
nums = [1, 2, 3, 4, 5]
doubled = list(map(lambda x: x*2, nums))   # [2,4,6,8,10]
evens   = list(filter(lambda x: x%2==0, nums)) # [2,4]
05 — Scope

Область видимости (LEGB)

Python ищет переменные по правилу LEGB: Local → Enclosing → Global → Built-in.

УровеньГде живётПример
LocalВнутри функциипеременная внутри def
EnclosingВнешняя функция (замыкание)переменная внешнего def
GlobalМодуль (файл)переменная в начале файла
Built-inPythonlen, print, range…
scope.py 
x = 10   # global

def outer():
    x = 20  # enclosing
    def inner():
        print(x)  # берёт из enclosing → 20
    inner()

outer()
print(x)   # global → 10
06 — Следующий шаг

Декораторы — что дальше

Декоратор — функция, которая принимает другую функцию и расширяет её поведение, не изменяя код.

decorator.py 
def timer(func):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"Время: {time.time()-start:.4f}с")
        return result
    return wrapper

@timer
def slow_sum(n):
    return sum(range(n))

slow_sum(1_000_000)   # Время: 0.0231с
